| 软件 | 版本 | 说明 |

| JDK | 21 | 必须使用 JDK 21 |

| IntelliJ IDEA | 2025.1.2 | 后端开发IDE |

## 二、JDK 21 安装与配置

### 2.1 下载 JDK 21

**方式一：Oracle JDK**
- 访问：https://www.oracle.com/java/technologies/downloads/#java21
- 选择 Windows x64 Installer 下载安装

**方式二：OpenJDK（推荐）**
- 访问：https://adoptium.net/
- 选择 Temurin 21 (LTS) 版本下载
- 或使用 scoop 安装：`scoop install temurin21-jdk`

### 2.2 配置环境变量

1. **设置 JAVA_HOME**
   ```
   变量名：JAVA_HOME
   变量值：C:\Program Files\Java\jdk-21（根据实际安装路径修改）
   ```

2. **添加到 Path**
   ```
   变量名：Path
   变量值：%JAVA_HOME%\bin
   ```

3. **验证安装**
   ```powershell
   java -version
   # 输出应显示：java version "21.x.x"
   
   javac -version
   # 输出应显示：javac 21.x.x
   ```

## 三、IntelliJ IDEA 配置

### 3.1 配置 JDK 21

1. 打开 IntelliJ IDEA
2. 进入 **File → Project Structure** (快捷键 `Ctrl+Alt+Shift+S`)
3. 选择 **Project**：
   - **SDK**: 点击下拉框 → **Add SDK** → **JDK**
   - 选择 JDK 21 的安装目录
   - **Language level**: 选择 **21 - Record patterns, pattern matching for switch**

4. 选择 **Modules**：
   - 确保 **Language level** 也是 **21**

### 3.2 配置 Maven

1. 进入 **File → Settings** (快捷键 `Ctrl+Alt+S`)
2. 导航到 **Build, Execution, Deployment → Build Tools → Maven（构建、执行、部署 → 构建工具 → Maven）**
3. 配置：
   - **Maven home path**: 选择 Maven 安装目录
   - **User settings file**: 选择 Maven 的 settings.xml
   - **Local repository**: 设置本地仓库路径

3. 导航到 **Maven → Runner（Maven → 运行器）**
   - **JRE**: 选择 **Use Project JDK (21)（使用项目 JDK (21)）**

### 3.3 启用 Lombok 注解处理

1. 进入 **File → Settings（文件 → 设置）**
2. 导航到 **Build, Execution, Deployment → Compiler → Annotation Processors（构建、执行、部署 → 编译器 → 注解处理器）**
3. 勾选 **Enable annotation processing（启用注解处理）**

### 3.4 安装必要插件

进入 **File → Settings → Plugins（文件 → 设置 → 插件）**，确保已安装：
- **Lombok** - 必须安装
- **Spring Boot Helper** - 推荐安装
- **MyBatisX** - 推荐安装

## 四、数据库配置

### 4.1 创建数据库

1. 启动 MySQL 服务
2. 执行数据库初始化脚本：
   ```powershell
   mysql -u root -p < database/init.sql
   ```
   
   或在 MySQL 客户端中执行：
   ```sql
   source e:/traezhuanyong/jiazheng/database/init.sql
   ```

### 4.2 修改数据库连接配置

编辑 `homeservice-backend/src/main/resources/application.yml`：

```yaml
spring:
  datasource:
    url: jdbc:mysql://localhost:3306/homeservice?useUnicode=true&characterEncoding=utf-8&serverTimezone=Asia/Shanghai&useSSL=false&allowPublicKeyRetrieval=true
    username: root
    password: 你的MySQL密码  # 修改为你的实际密码
```

## 五、启动项目

### 5.1 启动后端

**方式一：使用 IDEA 运行**

1. 打开 IDEA，选择 **File → Open**
2. 选择 `homeservice-backend` 目录
3. 等待 Maven 自动下载依赖（首次需要较长时间）
4. 找到 `HomeserviceApplication.java`
5. 右键 → **Run 'HomeserviceApplication'**

**方式二：使用命令行**

```powershell
cd homeservice-backend
mvn clean install -DskipTests
mvn spring-boot:run
```

**启动成功标志：**
```
Started HomeserviceApplication in x.xxx seconds
```

**访问 API 文档：**
- 地址：http://localhost:8080/doc.html

### 5.2 启动前端

```powershell
cd homeservice-frontend
npm install
npm run dev
```

**访问前端：**
- 地址：http://localhost:5173

## 六、常见问题解决

### 6.1 JDK 版本不匹配

**错误信息：**
```
java: invalid source release: 21
```

**解决方案：**
1. 确保安装了 JDK 21
2. IDEA 中配置 Project SDK 为 JDK 21
3. 检查 `pom.xml` 中的 `<java.version>21</java.version>`

### 6.2 Maven 依赖下载失败

**解决方案：**
1. 检查网络连接
2. 配置国内镜像源（阿里云）：
   
   编辑 Maven 的 `settings.xml`：
   ```xml
   <mirror>
       <id>aliyun</id>
       <mirrorOf>central</mirrorOf>
       <name>Aliyun Maven Mirror</name>
       <url>https://maven.aliyun.com/repository/public</url>
   </mirror>
   ```

### 6.3 数据库连接失败

**错误信息：**
```
Communications link failure
```

**解决方案：**
1. 确认 MySQL 服务已启动
2. 检查用户名密码是否正确
3. 检查数据库 `homeservice` 是否已创建

### 6.4 Lombok 不生效

**解决方案：**
1. 确保安装了 Lombok 插件
2. 启用注解处理：**Settings → Compiler → Annotation Processors → Enable**
3. 重启 IDEA

### 6.5 端口被占用

**错误信息：**
```
Web server failed to start. Port 8080 was already in use.
```

**解决方案：**
1. 查找占用端口的进程：
   ```powershell
   netstat -ano | findstr :8080
   ```
2. 结束进程或修改 `application.yml` 中的端口号

## 九、开发建议

1. **首次运行**：先启动后端，再启动前端
2. **热部署**：后端可添加 spring-boot-devtools 实现热部署
3. **调试模式**：IDEA 中使用 Debug 模式启动，可设置断点调试
4. **日志查看**：后端日志在控制台输出，前端日志在浏览器开发者工具

---

> **提示**：如遇到其他问题，请检查 IDEA 的 Event Log 和控制台输出信息。
